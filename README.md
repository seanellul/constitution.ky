# Constitution.ky - Interactive Constitution of the Cayman Islands

An interactive web application for exploring and understanding the Constitution of the Cayman Islands (Constitution Order 2009).

## Features

- **Interactive Navigation** - Browse through 9 constitutional parts and 125 sections
- **Advanced Search** - Full-text search across all constitutional content with relevance scoring
- **Legal Cross-References** - Understand connections between different sections
- **Visual Data Insights** - Interactive charts and analytics
- **Responsive Design** - Optimized for all devices
- **Dark Mode** - Full dark mode support
- **PWA Support** - Install as a native app

## Tech Stack

- **Framework**: Next.js with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL with Prisma ORM
- **Animations**: Framer Motion
- **Analytics**: Vercel Analytics + Custom anonymous tracking
- **Deployment**: Vercel

## Getting Started

```bash
npm install
npm run dev
```

## Constitutional Structure

The Cayman Islands Constitution Order 2009 comprises:

1. **Part I** - Bill of Rights, Freedoms and Responsibilities (Sections 1-28)
2. **Part II** - The Governor (Sections 29-42)
3. **Part III** - The Executive (Sections 43-58)
4. **Part IV** - The Legislature (Sections 59-93)
5. **Part V** - The Judicature (Sections 94-107)
6. **Part VI** - The Public Service (Sections 108-110)
7. **Part VII** - Finance (Sections 111-115)
8. **Part VIII** - Institutions Supporting Democracy (Sections 116-122)
9. **Part IX** - Miscellaneous (Sections 123-125)
